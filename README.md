# 欧路词典 Alfred Workflows 插件

## 使用
Alfred 激活后 ``di`` 调用打开欧路词典

```
di english
```

# 欧路词典 Alfred workflow 版本2

## 使用
1. 双击Eudic-lookup.alfredworkflow导入workflow
2. 使用 `di` 激活workflow
3. 系统提示选择对应的app，根据你自己选择是free版本还是专业版
4. 铛铛铛 `di future` 可以用了
